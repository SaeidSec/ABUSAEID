The Backbone

Core Architecture

Distributed intelligence across a multi-tier, pod-architecture cluster.

üîÄ Wazuh Cluster

Hybrid Master/Worker arrangement with horizontal event scaling and Filebeat 9.3.0 compatibility layer.

üíæ Data Persistence

3-Node OpenSearch cluster with quorum consensus, RBAC, and 99.9% data durability via the mTLS proxy.

‚ö° High Availability

Kubernetes-inspired pod architecture ‚Äî pause containers, dual Nginx LBs, Keepalived VRRP, and Telegraf metrics sidecar.

üîí Zero-Trust Proxy

FastAPI v3.0 microservice with mTLS enforcement, policy engine, rate limiting (5000 req/min), and round-robin health-checked upstreams.

üö® Incident Response

DFIR-IRIS platform for case management with Celery workers, RabbitMQ broker, and automated alert-to-case escalation from Wazuh.

Network Tier

Isolated L2 bridge network (172.25.0.0/16) with static IPs, managed by Keepalived for VIP failover and pod-level network sharing.

Master Manager

Orchestrates API access, agent enrollment, rule matching, honeypot log ingestion, Suricata IDS data, Trivy scans, and DFIR-IRIS case creation.

Log Management

Graylog 7.0 with MongoDB backend, connected to all 3 OpenSearch indexers with SSL truststore for unified log analytics.

Container Mgmt

Dockhand provides real-time Docker container management via web UI with direct socket access for operational visibility.

Engineering Masterpiece

FastAPI mTLS Proxy v3.0

A production-grade Python microservice bridging Wazuh Managers and OpenSearch 2.x clusters. Features include mutual TLS authentication, YAML-based policy engine with request filtering, automated health checking with round-robin load balancing, real-time config hot-reload via filesystem watcher, bulk request transformation (removing legacy _type fields), gzip decompression/recompression, Elasticsearch license mocking, and rate limiting at 5,000 req/min.

FastAPI mTLS Policy Engine Health Checking Round-Robin LB Rate Limiting Hot-Reload Gzip Transform License Mocking HTTPX Async
Compatibility Layer

Filebeat 9.3.0 Upgrade

Custom compatibility shim to bridge legacy Wazuh managers with modern Filebeat 9.3.0. Includes an opensearch_proxy.py script that mocks Elastic license heartbeats and translates 7.x CLI arguments to 9.x signatures, ensuring flawless data ingestion in bleeding-edge environments.
