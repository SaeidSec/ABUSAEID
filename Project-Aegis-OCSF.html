Engineering Deep Dive

Core Architecture

A modular, multi-threaded engine built for extreme efficiency and low-latency security analytics.

🦀 Rust ETL Engine

Leveraging Rust's zero-cost abstractions and memory safety to process 100k+ events/sec with minimal CPU overhead.

🏛️ ClickHouse MergeTree

Utilizing columnar storage and specialized codecs (ZSTD/Delta) to achieve massive compression and sub-second SQL performance.

🛡️ OCSF 1.7.0 Mapping

Automatically mapping Wazuh's data.win.eventdata and rule.description fields to standardized OCSF Security Control classes.

📊 SQL-Native Grafana

Direct SQL integration allowing for complex threat hunting queries (JOINs, Window Functions) that are impossible in OpenSearch.

⚡

Parallel Processing

Multi-threaded ingestion using Rayon and Tokio to ensure the pipeline never bottlenecks, even during massive log bursts.

📦

Batch Ingestion

Optimized ClickHouse batching (100k rows/block) to maximize disk I/O throughput and minimize table fragmentation.
